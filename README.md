# 🖤 SanityPress (FORK'D!)

Shoutout to @nuotsu and everyone that has contributed to the [nuotsu/sanitypress](https://github.com/nuotsu/sanitypress) project!

**_NOTE:_** If you're interested in this project please check out the original repo [here](https://github.com/nuotsu/sanitypress)

This is a forked version of that project that I am working on in my free time. The sanitypress team is definatley more experienced with [Next.js](https://nextjs.org/) and [Sanity.io](https://www.sanity.io/) than myself and they will have more up to date versions with any fixes & more. I'm mainly focused on adding different styling and custom modules (such as my Timeline module!) but my revisions require more node_modules than the simplier, original version.

## Biggest Changes from the Original Project

- Themes systematically handled via DaisyUI
- Light/Dark mode toggle applied directly to the header
- A vertical Timeline module I created by revising the [Aceternity UI, Timeline Component](https://ui.aceternity.com/components/timeline) so the context can be handled through the Sanity Studio. _(Still working on alignment for different imag sizes and screen sizes but it works pretty well!)_
- A custom "Cert List" module that allows you to display your certification badges with a animated tooltip _(Still need to adjust css to perfect the tooltip positioning no matter the length of cert name)_

## Working on

- A GitHub integration in order to pull personal profile data as well as some recent repositories you may be working on _(This is currently only a concept that I have not been able to get working as I would like)_

Main reason for these modifications was to host this as a personal portfolio/blog website. Becuase some of my decisions to use unecessary node modules, I will not be adding any pull requests to the original sanitypress project.
